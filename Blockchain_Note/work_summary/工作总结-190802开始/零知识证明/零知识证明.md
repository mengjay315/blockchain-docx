# 零知识证明

通用的零知识证明的优势分为两方面，其一是**隐私性**，你可以在验证数据的同时保证不公开数据；第二是**可扩展性**，你验证的程序容量可能很大，但验证的过程可以是很快的。

通过零知识证明，以太坊的新技术Rollup能够将每秒的交易量从30笔增加到4000笔。
