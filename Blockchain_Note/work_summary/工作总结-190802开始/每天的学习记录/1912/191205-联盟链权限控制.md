加入主cosmos主网的步骤：

1. 本机初始化一个节点，gaiad init <节点名字> --chain-id <主网链id>
2. copy主网的创世文件，覆盖自己生成的创世文件
3. 在配置文件中配置主网上活跃的种子节点
4. gaiad start (启动一个全节点)

退出主网，也就是把start进程给杀死。

也就是说cosmos网络，节点是随意加入和退出的，并且立即生效。

加入主网后成为网络中的一个节点，这时只是普通的节点，可以通过质押一定量的token申请成为验证人，也可以委托token给验证人节点，
