I created two validator nodes, and creating multiple nodes is similar. You can try it, too.

## Run first second node on your machine 

```bash
nsd init test01 --chain-id namechain

nscli keys add jack

nscli keys add alice

# Add both accounts, with coins to the genesis file
nsd add-genesis-account $(nscli keys show jack -a) 10000000000000nametoken,1000000000000000000stake
nsd add-genesis-account $(nscli keys show alice -a) 10000000000000nametoken,100000000000000000stake

# Configure your CLI to eliminate need for chain-id flag
nscli config chain-id namechain
nscli config output json
nscli config indent true
nscli config trust-node true

nsd gentx --name jack 

nsd collect-gentxs

and to make sure your genesis file is correct, run:

nsd validate-genesis

set minimum-gas-prices (.nsd/config/app.toml)
minimum-gas-prices = "0.02nametoken

nsd start
```

this command `nsd gentx --name jack is an alias of the 'tx create-validator' command'

You have run your first node successfully，you can view validator info

## Run second node on another machine

```bash
nsd init test02 --chain-id namechain
nscli keys add tom

```

#### overwrite ~/.nsd/config/genesis.json with first node's genesis.json

#### change persistent_peers

#### **Note:** 

If you are running the third or more nodes, the `persistent peers`of config.toml of your current node, here you can fill in other nodes in the current chain, separated by commas, and write several more.

```bash
vim /.nsd/config/config.toml
persistent_peers = "id@first_node_ip:26656"
run "nscli status" on first node to get id.
```

The first node alice transfer token to the secon node'tom

```bash
nscli tx send $(nscli keys show alice -a) (tom'address) 100000000000nametoken --gas-prices 0.06nametoken --from alice
nscli tx send $(nscli keys show alice -a) (tom'address) 10000000000stake --gas-prices 0.06nametoken --from alice

```

## start this second node

```bash
nsd start
```

Create one validator of namechain on your second node

```bash
nscli tx staking create-validator \
  --amount=100000000stake \
  --pubkey=$(nsd tendermint show-validator) \
  --moniker="test02" \
  --chain-id=namechain \
  --commission-rate="0.10" \
  --commission-max-rate="0.20" \
  --commission-max-change-rate="0.01" \
  --min-self-delegation="1" \
  --gas="auto" \
  --gas-prices="0.30nametoken" \
  --from=tom
```
