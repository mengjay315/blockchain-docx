## 区块链浏览器

1. ### 登录服务器

ssh xtsxx@172.168.0.78, 密码：123456

2. ### 数据库操作

- 登录 postgresql数据库

  ./initdb /usr/local/var/postgres (初始化)

  ./psql -h 127.0.0.1 -p 5432 -d postgres -U xtsxx （xtsxx为服务器用户名， -d默认登陆的数据库）

  ./psql -U xtsxx -d postgres

- 查看配置文件

  cat /usr/local/var/postgres/postgresql.conf

- 查看数据的目录

  show data_directory;

  /usr/local/var/postgres

- 数据库重启

  ./pg_ctl -D /usr/local/var/postgres restart

- 查询数据库活跃连接数

  select count(1) from pg_stat_activity;

- 查询最大连接数

  show max_connections;

- 查询超级用户连接数

  show superuser_reserved_connections;

- 查看数据库进程

  ps -ef | grep postgres
