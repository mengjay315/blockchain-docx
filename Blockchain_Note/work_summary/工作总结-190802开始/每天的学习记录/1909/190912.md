部署合约是一个交易，会调用以太坊的`sendTransaction`方法，

```
txhash = eth.sendTansaction({from:primary, data: code})
```

`from`是部署合约的地址，`data`是**合约编译后的字节码**。

```
contractaddress = eth.getTransactionReceipt(txhash);
```

调用`getTransactionReceipt`接口，传一个`txhash`,会返回一个 `transaction receipt object对象`.

```bash

// Result
```
